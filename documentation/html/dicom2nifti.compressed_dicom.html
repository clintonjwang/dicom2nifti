dicom2nifti.compressed_dicom.is_dicom_file(filename)[source]¶

Util function to check if file is a dicom file the first 128 bytes are preamble the next 4 bytes should contain DICM otherwise it is not a dicom

Parameters:filename (six.string_types) – file to check for the DICM header block
Returns:True if it is a dicom file
